We Are

ALFA, the Automated Log File Analyser. ALFA is a Single-Page Application used to scan and analyse log files then intelligently provide suggestions or potential solutions to issues encountered on the log files.

Upload Log File

Users can drag-and-drop (or select) one or multiple* log files to upload. The system will then use its knowledge-base database that stores articles of the encountered problems and intelligently search and retrieve the relevant articles that are potential solutions to the analyzed log file problems.

* A maximum of 5 log files
